# Python AI Chatbot

A simple AI chatbot built with Python that can understand user messages and respond using basic natural language processing techniques.

## Features

* Responds to user messages
* Uses basic natural language processing (NLP)
* Finds the most relevant response from the dataset
* Runs in the terminal

## Technologies Used

* Python
* scikit-learn
* pandas
* nltk
* numpy

## How It Works

The chatbot analyzes the user's input, processes the text using NLP techniques, and finds the most similar question in its dataset. Then it returns the corresponding answer.

## Installation

1. Clone the repository

```bash
git clone https://github.com/username/chatbot-ai
```

2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run the program

```bash
python main.py
```

```
You: Hello
Bot: Hi! How can I help you today?
```
